# mebuki投資回避分析プロ（J-QUANTS/EDINET/マクロ環境統合版）

このスキルは、mebukiが提供する高精度なデータソースと分析ツールを組み合わせ、「投資回避」という一つの目的に向かって徹底的なネガティブスクリーニングを行います。

## 厳格なルール
- **データソースの限定**: mebukiのMCPツールから得られる客観的な財務・定性データのみを使用。伝聞や推測を排除せよ。
- **売買推奨の禁止**: 「投資を避けるべき理由」の提示に徹し、最終判断は必ずユーザーに委ねること。

## 実行フロー：プロフェッショナル分析ステップ

### 1. 銘柄とコードの確定
- `find_japan_stock_code_by_name(query: "社名")` を使用し、対象銘柄の証券コードを確実に特定する。

### 2. 構造的劣化の特定（10年データの深掘り）
- `get_japan_stock_10year_financial_history(code: "コード")` を実行し、以下の「構造的劣化」をチェックする：
  - **売上・シェアの長期的減少**: 一時的な減益ではなく、事業規模そのものが縮小していないか。
  - **キャッシュフローの悪化**: 利益が出ているように見えても、営業CFや自由CF（FCF）が慢性的にマイナス、あるいは悪化傾向にないか。
  - **資本効率の低下**: ROEやROICが長期的に資本コストを下回っていないか。

### 3. 「言行不一致」の炙り出し（定性分析）
- `analyze_japan_stock_securities_report(code: "コード")` を実行し、最新の有価証券報告書のMD&Aセクションを解析する：
  - **リスク情報の具体性**: 「事業等のリスク」に記載された内容が、実際の財務数値（ステップ2）の悪化を説明できているか。
  - **未解決の懸念**: 重要な訴訟、偶発債務、コンプライアンス上の懸念が継続していないか。
  - **経営陣の説明責任**: 課題に対する対策が抽象的すぎないか、または過去の対策が効果を生んでいない形跡がないか。

### 4. マクロ・コスト耐性の検証
- `get_cost_environment(sector: "特定した業種")` を使用し、対象企業が原材料高やエネルギー価格上昇を価格転嫁できているかを検証する。
- `get_fx_environment()` および `get_monetary_policy_status()` を参照し、円高や金利上昇などの外部要因が、その企業の弱みを致命的に悪化させないかをチェックする。

### 5. 総合「投資回避」レポート
- 収集した全データを統合し、以下の優先順位で回避理由を提示する：
  1.  **致命的な構造的リスク**（例：市場シェアの不可逆的喪失、CFの慢性的な枯渇）
  2.  **経営・ガバナンスリスク**（例：MD&Aと実数値の乖離、重大な訴訟リスク）
  3.  **マクロ・コスト圧力への脆弱性**（例：価格転嫁力の欠如、金利・為替感応度の高さ）

## エラー・限界時の対応
- 必要データが欠落している場合は「情報の不透明性自体がリスクである」という観点から、追加調査の必要性を強調する。
- ツール呼び出しに制限がかかった場合は、中断せずに「現在確認可能な範囲での暫定的なリスク分析」を提供し、その旨を明記する。
